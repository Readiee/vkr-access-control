# Приложения к отчёту

В отчёт включены шесть приложений. Каждое содержит вспомогательный материал, на который ссылается основная часть.

| Литера | Содержание | Источник в репозитории |
|---|---|---|
| А | Диаграмма Use Case (UC-1..UC-10) | [diagrams/uml/use-cases.pz.mmd](../../diagrams/uml/use-cases.pz.mmd) |
| Б | Полные таблицы функциональных и нефункциональных требований | Раздел 1 отчёта (расширенная версия) |
| В | C4-диаграммы (контекст, контейнеры), диаграмма классов TBox, sequence-диаграммы | [diagrams/c4/](../../diagrams/c4/), [diagrams/uml/class-tbox.pz.mmd](../../diagrams/uml/class-tbox.pz.mmd), [diagrams/uml/sequence-*](../../diagrams/uml/) |
| Г | Маппинг классов онтологии на сущности пяти СДО | Таблица соответствий по доменным концептам |
| Д | Фрагмент SWRL-каталога с примерами правил по типам политик | [code/onto/scripts/](../../code/onto/scripts/) |
| Е | Скриншоты пользовательского интерфейса | По итогам прогона `code/frontend/` в режиме разработки |

При сборке отчётного документа каждое приложение оформляется на отдельной странице с заголовком вида «ПРИЛОЖЕНИЕ А. Диаграмма вариантов использования». Изображения экспортируются в формате PNG или SVG из соответствующего исходника. Для диаграмм Mermaid и PlantUML используются `mmdc` и `plantuml` соответственно; результаты сохраняются в `practice_report/appendices/figures/`.
