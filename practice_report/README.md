Итого основной части: около 13 страниц. С титулом, индивидуальным заданием и содержанием выходит 16–17 страниц; общий объём с приложениями укладывается в 20 страниц.

Стиль текста соответствует требованиям [docs/SAT_PZ_STYLE.md](../docs/SAT_PZ_STYLE.md) — без AI-маркеров, тире-связок, штампов «представляет собой», «играет ключевую роль», «таким образом», «более того», «важно отметить»; перечисления вне ровно трёх элементов; кавычки «ёлочками» при сборке итогового .docx.

При финальной сборке отчёта в Word потребуется:

1. Прогнать поиск-замену прямых кавычек на «ёлочки».
2. Заменить дефисы на длинные тире там, где Word не сделал автозамену.
3. Сконвертировать markdown-списки в нумерованные по ГОСТ.
4. Вставить экспортированные диаграммы и скриншоты в соответствующие приложения.
